# test_acme
## Front end

Goto test_acme_front directory and open console
Run following command
### Install packages
      yarn install
### Run front end
      yarn serve
### Build front end
      yarn build
### Test unit
      yarn test:unit
### Build docker image

      docker-compose up
  
  
 -start docker image
 
 
          docker-compose start
 -stop docker image
 
 
 
          docker-compose stop
